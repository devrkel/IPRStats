Open an XML file

Launches a file-chooser dialog for the user to select the InterProScan output XML file or IPRStats file to be opened. It then calls the corresponding function to open it and refresh the GUI data.

OnSaveAs(self, event)

source code 

Saves the current session.

Compresseses the current session folder into a .tar.bz2 so that the user doesn't have to deal with parsing and querying an XML file every time.

OnExportHTML(self, event)

source code 

Exports current session as HTML

Takes the parsed and queried data from an opened XML file and writes it as HTML. Each app from InterProScan gets its own static HTML page.

OnExportXLS(self, event)

source code 

Save current session as a spreadsheet

Uses python-xlwt to write the parsed XML file as a spreadsheet file compatible with OpenOffice.org and Microsoft Excel
